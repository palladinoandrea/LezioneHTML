## Struttura base di un documento HTML

- `<!DOCTYPE html>`  
  Indica che il documento è scritto con lo standard **HTML5**.

- `<html></html>`  
  È l’elemento **radice** della pagina e racchiude tutto il contenuto del documento.

- `<head></head>`  
  Contiene i **metadati**, ovvero informazioni non visibili direttamente all’utente ma fondamentali per il browser e i motori di ricerca.

### Contenuto del tag `<head>`

- `<title></title>`  
  Definisce il **titolo della pagina**, visualizzato nella scheda del browser.  
  È **obbligatorio** in ogni documento HTML.

- `<meta charset="UTF-8">`  
  Specifica la **codifica dei caratteri** del documento.  
  Ad esempio, UTF-8 consente la corretta visualizzazione dei caratteri speciali.

- `<meta name="description">`  
  Fornisce una **breve descrizione** del contenuto della pagina, utile per i motori di ricerca.

- `<meta name="keywords">`  
  Indica le **parole chiave** associate alla pagina.

- `<meta name="author">`  
  Specifica l’**autore** del documento HTML.

- `<meta name="viewport">`  
  Imposta la corretta visualizzazione della pagina sui **dispositivi mobili**, rendendo il layout responsive.

- `<body></body>`  
  La sezione `<body>` contiene tutti i **contenuti visibili** della pagina web  
  (testo, immagini, link, ecc.).
